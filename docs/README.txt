This documentation is generated using Sphinx.

You should be able to read it at https://sciath.rtfd.io

If you want to build it locally, you will need Python, Sphinx and a theme for
Read the Docs. For example, with conda you could run

    conda install sphinx sphinx_rtd_theme

Then, from this directory,

    make html

Then, open with your web browser, e.g.

    firefox _build/html/index.html
